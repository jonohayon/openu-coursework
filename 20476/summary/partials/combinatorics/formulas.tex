\subsection*{עקרון ההכלה וההפרדה}
תהי $U$ קבוצת עולם, $A_{1}, \ldots, A_{n} \subseteq U$ קבוצות סופיות. נסמן:
\[
S_{i} = \sum^{i}_{1 \le k_{1} \le \ldots \le k_{n} \le n} |\bigcap^{n}_{i=1} A_{i}|
\]
כלומר $S_{1} = \sum |A_{i}|, S_{2} = \sum_{1 \le i \le j} |A_{i} \cap A_{j}|$ וכו'. אזי לפי עקרון ההכלה וההפרדה נוכל לחשב את עוצמת איחודן של $A_{i}$ באופן הבא:
\[
|A_{1} \cup \ldots \cup A_{n}| = |\bigcup^{n}_{i=1} A_{i}| = \sum^{n}_{i=1} (-1)^{i-1} S_{i} = S_{1} - S_{2} + S_{3} - S_{4} + \ldots + (-1)^{n-1}S_{n}
\]
צורת רישום אחרת )מציאת עוצמת חיתוך המשלימים של $A_{i}$ לפי $U$(:
\[
|A_{i}' \cap \ldots \cap A_{n}'| = |U| - \sum^{n}_{i=1} (-1)^{i-1} S_{i}
\]
